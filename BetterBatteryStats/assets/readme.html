BetterBatteryStatsRelease Notes

v1.9.0.0RC6 2012-07-14

  • Merge packages in eventX-YYYY kernel wakelocks instead of taking current list
  • Fixed missing CRLF in dumpfile

v1.9.0.0RC5 2012-07-12

  • Fixed bug introduced with package-listing for eventX-YYYY kernel wakelocks
  • Fixed intermittent inconsistency in the sum of percents "deep sleep" and "awake" in "other" stats
  • Add kernel version to dumpfile

v1.9.0.0RC4 2012-07-11

  • Fixed inconsistent behavior partial wakelocks (not showing the right data, depending on stats type)
  • Fixed handling of 'eventX-YYYY' kernel wakelock (decodes the process name)
  • Fixed (again) an error in the calculated % in network stats (total volume was correct but sometimes the % was too small)
  • Added raw network stats
  • Widgets now are always refreshed upon screen on. Preferences allow to disable timed refresh

v1.9.0.0RC3 2012-07-09

  • References will be deleted on first start of the app
  • Fixed an error in the calculated % in network stats (total volume was correct but sometimes the % was too small)
  • Fixed error leading to and ANR when turning the screen on or after unplugging USB
  • Fixed error where "since unplugged" or "since screen off" references didn't seem to be saved
  • Fixed error where "Deep sleep" value in "Other" stats was wrong (too high)

v1.9.0.0RC2 2012-07-06

  • Default stats and stat types are being migrated to new default values if you come from a pre-1.9 release. Please go to the preferences to change them to suit your needs
  • Added perm android.permission.BLUETOOTH to solve the problem of unavailable 'batteryinfo' service on some ROM/Kernel combos
  • Fixed "since screen off" being always called when clicking widgets (Please reset your default stat/stat type preferences!)
  • Removed "test" menu
  • Fixed stats showing no data

v1.9.0.0RC1 2012-07-05

  • Added network usage stats
  • Added CPU states stats
  • "other" view can now be customized (cleaned up) by selecting the values that should be displayed (does not affect dump)
  • Removed widget logging to SD
  • Fixed graph layouts for hires screens
  • Rewrote the how-to for more clarity
  • Fixed icon sizes
  • Added fallback stat type for widgets (in case the selected one has no reference)
  • Stat and stat type displayed when tapping the widgets can be defined in preferences
  • Battery voltage and level frop added to dumpfile
  • Fixed disappearing "since..." time
  • Changed stat order for more clarity
